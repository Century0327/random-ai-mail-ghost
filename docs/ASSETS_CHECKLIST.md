> 按优先级排序，P0 = 必须，P1 = 重要，P2 = 锦上添花。

## 十一、制作建议

### 工具推荐

| 用途 | 推荐工具 |
|------|----------|
| 像素画 | Aseprite（付费，行业标准）、Pixilart（免费在线）、Procreate（iPad） |
| 图标/Logo | Figma（免费）、Adobe Illustrator |
| 动画 | Aseprite（自带动画）、Spine（2D骨骼动画，适合桌宠） |
| APNG 制作 | Aseprite 导出 APNG、或 PNG 序列 + ffmpeg |
| 截图/录屏 | OBS（免费）、ShareX（Windows 免费） |

### 风格统一检查清单

- [ ] 所有角色同画风（像素风/手绘风/扁平风）
- [ ] 所有物品同视角（等距/正面/侧面）
- [ ] 所有图标同尺寸规范（商店 128×128、成就 64×64、角色 512×512）
- [ ] 所有透明底 PNG 正确导出 Alpha 通道
- [ ] Steam Capsule 图无透明底、无禁止元素（定价、平台 logo）
- [ ] 中文字体在 Steam 截图中清晰可读（建议用粗体像素字如"像素字体""ZCOOL"系列）
